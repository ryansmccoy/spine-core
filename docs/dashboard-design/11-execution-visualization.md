# Execution & Timeline Visualization

> Part of: [Dashboard Design](00-index.md)

## Purpose

Visualizations help operators understand:
- **Sequence**: What ran when, in what order
- **Dependencies**: What depends on what
- **Duration**: How long things take
- **Anomalies**: What's unusual about this run
