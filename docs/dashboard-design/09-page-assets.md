## Data Tier Tabs

### OTC Tab

| Field | Description |
|-------|-------------|
| Description | FINRA OTC Transparency data |
| Source | FINRA weekly publications |
| Frequency | Weekly (Friday publish) |
| Latency | T+5 business days |
| History | 52 weeks rolling |

### NMS Tier 1 Tab

| Field | Description |
|-------|-------------|
| Description | NMS stocks with high volume |
| Source | FINRA NMS Transparency |
| Frequency | Weekly |
| Criteria | >$2B market cap |

### NMS Tier 2 Tab

| Field | Description |
|-------|-------------|
| Description | NMS stocks with lower volume |
| Source | FINRA NMS Transparency |
| Frequency | Weekly |
| Criteria | <$2B market cap |

### Derived Analytics Tab (Intermediate+)

| Dataset | Description |
|---------|-------------|
| `volume_6w_avg` | 6-week rolling average volume |
| `wow_change` | Week-over-week volume change |
| `top_movers` | Top 100 by volume rank change |
| `sector_aggregates` | Volume by sector |

- Download CSV

- Week-over-week changes
- Volume trends
- Symbol history charts

- Data lineage links
- Capture_id filtering
- Point-in-time queries
- Retention management
